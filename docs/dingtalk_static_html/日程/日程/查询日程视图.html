查询日程视图

调用本接口，查询某用户在某段时间内的日程列表，可以获取到日程标题、日程的开始日期、开始时间、结束日期、结束时间等信息。

接口调用说明

  • 本接口与查询日程列表接口的区别在于对循环日程的处理。本接口会根据循环规则展开查询区间内的所有循环日程，但不会实际在数据库中生成对应实例。 例如，有一个从1.1号开始的每日循环日程，查询时已经是1.4号，查询区间为1.1号至1.10号，那么本接口会返回1.1～1.10号的所有循环日程，而查询日程列表接口是不会返回1.4号之后的日程。

  • 可以根据返回的日程对象seriesMasterId是否为空，来判断该日程是否为一个循环日程的实例。
