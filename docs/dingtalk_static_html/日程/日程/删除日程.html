删除日程

调用本接口，删除指定日程。

接口调用说明

  • 当组织者删除日程时（url中使用组织者uninoId），所有参与者会收到日程取消通知，同时日程将从所有参与者的日历中删除。

  • 当参与者删除日程时（url中使用参与者uninoId），仅将日程从自己的日历中删除，其他日程参与者不受影响。

请求

基本信息

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/calendar/users/{userId}/calendars/{calendarId}/events/{eventId}

HTTP Method

支持的应用类型

appType-企业内部应用　appType-第三方企业应用　appType-第三方个人应用

权限要求

permission-Calendar.Event.Write-日历应用中日程写权限

  • 第三方个人应用，调用获取用户token接口获取。
