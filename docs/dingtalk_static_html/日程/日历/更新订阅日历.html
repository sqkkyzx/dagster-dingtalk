Array of String

否

  • 企业内部应用和第三方企业应用，调用查询用户详情接口获取unionid参数值。

  • 第三方个人应用，调用获取用户通讯录个人信息接口获取unionId参数值。

openConversationIds

Array of String

否

  • 第三方个人应用无需传此参数。
