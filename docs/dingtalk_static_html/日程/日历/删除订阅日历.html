删除订阅日历

调用本接口，删除订阅日历。

接口调用说明

  • 目前仅允许日历创建者删除日历，因此Path参数中的userId字段必须是日历创建者的unionId。

  • 订阅日历被删除后，已订阅该日历的用户会自动解除订阅。

请求

基本信息

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/calendar/users/{userId}/subscribedCalendars/{calendarId}

HTTP Method

DELETE

支持的应用类型

appType-企业内部应用　appType-第三方企业应用　appType-第三方个人应用

权限要求

permission-Calendar.Calendar.Write-日历应用中日历写权限

请求头

名称

类型

是否必填

描述

x-acs-dingtalk-access-token

String

是

调用该接口的访问凭证，通过以下获取：

  • 企业内部应用，调用获取企业内部应用的accessToken接口获取。

  • 第三方企业应用，调用获取第三方应用授权企业的accessToken接口获取。

  • 第三方个人应用，调用获取用户token接口获取。

路径参数

名称

类型

是否必填

String

是

日历创建者的unionId。

  • 企业内部应用和第三方企业应用，调用查询用户详情接口获取unionid参数值。

  • 第三方个人应用，调用获取用户通讯录个人信息接口获取unionId参数值。
