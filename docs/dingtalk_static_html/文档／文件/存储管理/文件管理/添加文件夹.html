添加文件夹

调用本接口，在存储空间内添加文件夹。

接口调用说明

在添加存储空间后，可调用本接口在空间内添加文件夹。

  • 存储空间类型为USER时，空间拥有权限的用户都有操作权限，其他员工调用添加权限接口进行授权。

  • 存储空间类型为APP时，任何人操作都需要调用添加权限接口进行授权。
