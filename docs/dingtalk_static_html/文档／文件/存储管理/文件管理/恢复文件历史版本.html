恢复文件历史版本

调用本接口，恢复文件历史版本。

接口调用说明

例如，有一个文件名为测试文件.txt，目前最新版本为5。调用本接口可以把该文件恢复为第3版本。接口调用成功后，测试文件.txt的最新版本为6，文件内容与第3版本一致。

权限要求
