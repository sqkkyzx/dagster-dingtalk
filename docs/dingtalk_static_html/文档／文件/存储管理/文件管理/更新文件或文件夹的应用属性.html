更新文件或文件夹的应用属性

调用本接口，更新文件或文件夹的应用属性。

接口调用说明

接口传入的应用属性信息，会根据属性名称校验。

  • 如果文件或文件夹原有属性名与接口参数传入的属性名相同，接口执行更新属性。

  • 如果文件或文件夹原有属性名与接口参数传入的属性名不相同，接口执行添加属性。
