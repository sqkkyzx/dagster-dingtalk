删除文件或文件夹

调用本接口，可以实现删除文件或文件夹。

名称

类型

描述

x-acs-dingtalk-access-token

String

是

调用该接口的访问凭证，通过以下获取：

  • 企业内部应用，调用获取企业内部应用的accessToken接口获取。

  • 第三方企业应用，调用获取第三方应用授权企业的accessToken接口获取。

路径参数

名称

类型

是否必填

描述

spaceId

String

是

空间Id，可调用添加空间接口获取id参数值。

dentryId

String

是

需要被删除的文件或文件夹Id，可调用获取文件或文件夹列表或根据dentryUuid获取spaceId和dentryId接口获取id参数值。

查询参数

名称

类型

是否必填

描述

toRecycleBin

Boolean

否

删除后，是否备份到回收站：

  • true：是
  • false：否，默认值

目前仅支持新建空间接口创建的云盘空间内的文件和文件夹支持备份到回收站。

unionId

String

是

操作者unionId，可调用查询用户详情接口获取unionid参数值。
