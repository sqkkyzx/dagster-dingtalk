添加空间

调用本接口，在企业云盘下创建一个新的空间，在该空间内可以进行新建文件夹、上传文件等操作。

接口调用说明

存储空间介绍：

  • 存储空间属于企业云盘的一部分，但是调用本接口添加的空间在客户端云盘内是不可见的。

  • 存储空间会占用企业云盘的空间。

存储空间分类：

  • 用户空间，即USER空间。该空间的文件只能授予永久权限，要求授权者有管理权限才能授权。

  • 应用空间，即APP空间。该空间的文件只能授予临时权限，不要求授权者有管理权限。
