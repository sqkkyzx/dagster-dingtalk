初始化文件分片上传

调用本接口，初始化文件分片上传。

接口调用说明

接口调用流程，请参考下方步骤：

步骤一：调用本接口，初始化文件分片上传，获取上传标识uploadKey。

步骤二：调用获取文件分片上传信息接口，获取每片文件的上传信息。

步骤三：循环对每片文件执行上传操作。

步骤四：使用步骤一接口获取的上传标识uploadKey，调用提交文件接口，完成上传。
