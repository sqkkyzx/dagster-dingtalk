获取存储中异步任务信息

调用本接口，获取存储中异步任务信息，异步任务是否已完成、是否成功等。例如，批量移动文件或文件夹、批量拷贝文件或文件夹、批量删除文件或文件夹等操作，如果被操作对象存在子节点，对应的操作就是异步操作。

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/storage/tasks/{taskId}

HTTP Method

支持的应用类型

appType-企业内部应用　appType-第三方企业应用

权限要求
