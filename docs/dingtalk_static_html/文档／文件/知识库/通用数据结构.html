访问者对当前节点的权限等信息。

spaceActions

Array of String

PUBLIC_SHARE_SPACE,GET_SPACE

空间的操作列表。

  • PUBLIC_SHARE_SPACE：公开发布知识库

  • HANDOVER_WORKSPACE：移交空间

  • UPDATE_SPACE：更新空间基本信息

  • GET_SPACE：查询空间基本信息

  • GET_SPACE_SECURITY：查询空间安全设置

  • DELETE_SPACE：彻底删除空间

  • MANAGE_SPACE_SECURITY：管理空间安全设置

  • GET_SPACE_MEMBER：查询空间成员

  • MANAGE_SPACE_PIN：管理空间置顶

  • USE_SPACE_TEMPLATE：使用空间模板创建

  • RECYCLE_SPACE：归档空间

  • GET_SPACE_TEMPLATE：查询和预览空间模板

  • RECOVER_SPACE：恢复空间

  • MANAGE_SPACE_TEMPLATE：管理空间模板

  • MANAGE_SPACE_MEMBER：管理空间成员

  • RENAME_SPACE：重命名空间

dentryActions

Array of String

COPY_DENTRY, GET_DENTRY_MEMBER

节点的操作列表。

  • GET_DENTRY：查询节点基本信息

  • LIST_CHILD：列子节点

  • CREATE_CHILD：创建子节点

  • RENAME_DENTRY：重命名节点

  • MOVE_DENTRY：移动节点

  • COPY_DENTRY：复制节点

  • RECYCLE_DENTRY：回收节点

  • RECOVER_DENTRY：恢复节点

  • DELETE_DENTRY：删除节点

  • GET_DENTRY_MEMBER：获取节点成员列表

  • MANAGE_DENTRY_MEMBER：管理节点成员

  • PREVIEW：文档预览

  • CONTENT_COPY：内容复制

  • DOWNLOAD_ATTACHMENT：下载文档内附件/图片等资源

  • EXPORT：导出为其他格式

  • EXPORT_IMAGE：导出为长图

  • UPLOAD_ATTACHMENT：上传文档内附件、图片等资源

  • EDIT：文档编辑

  • READ_HISTORY：查看历史版本

  • EDIT_HISTORY：编辑历史版本

  • SAVE_AS：另存为

  • SAVE_AS_TEMPLATE：另存为模板

  • LINK_SHARE：链接或卡片分享

  • PUBLIC_SHARE：公开分享

  • CHANGE_PERMISSION_MODE：修改权限模式

  • TEMPORARY_SHARE_NODE：临时共享节点

  • DOWNLOAD：下载

  • UPLOAD：上传

  • SAVE_AS_QUICK_LINK：保存为快捷方式

  • SEND_COPY_FILE：发送副本

  • CLOUD_PRINT：云打印

  • MANAGE_SECURITY_SETTING：管理安全设置

  • UPLOAD_FOLDER：上传文件夹

  • IMPORT：导入

  • IMPORT_FOLDER：导入文件夹

roleCode

String

5

当前用户对这个空间的访问角色。

  • 0：无权限

  • 1：仅可查看

  • 2：可查看/下载

  • 3：可编辑

  • 4：可管理

  • 5：所有者
