获取知识库列表

调用本接口，根据操作人unionId，获取知识库列表信息。

字段

值

HTTP URL

https://api.dingtalk.com/v2.0/wiki/workspaces

HTTP Method

GET

支持的应用类型

appType-企业内部应用　appType-第三方企业应用

权限要求

permission-Wiki.Workspace.Read-知识库读权限

名称

类型

是否必填

描述

x-acs-dingtalk-access-token

String

是

调用该接口的访问凭证，通过以下获取：

  • 企业内部应用，调用获取企业内部应用的accessToken接口获取。

  • 第三方企业应用，调用获取第三方应用授权企业的accessToken接口获取。

查询参数

名称

类型

是否必填

描述

nextToken

String

否

分页游标, 首次拉取不用传。

maxResults

Integer

否

分页大小，默认值30。

最大值30。

orderBy

String

否

排序方式，枚举值:

  • VIEW_TIME_DESC：查看时间倒序

  • CREATE_TIME_DESC：创建时间倒序

  • MODIFIED_TIME_DESC：修改时间倒序

withPermissionRole

Boolean

否

是否同时查询权限：

  • true：查询

  • false：不查询

默认值false。

operatorId

String

是

操作人unionId，可通过调用查询用户详情接口获取。
