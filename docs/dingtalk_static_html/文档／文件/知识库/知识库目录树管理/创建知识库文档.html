创建知识库文档

调用本接口，在知识库内创建文档或者文件夹。

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/doc/workspaces/{workspaceId}/docs

HTTP Method

POST

支持的应用类型

appType-企业内部应用　appType-第三方企业应用

权限要求

permission-Document.WorkspaceDocument.Write-知识库文档写权限

请求体

名称

类型

是否必填

描述

name

String

是

文档名称。

docType

String

是

文档类型，取值：

  • DOC：文字

  • WORKBOOK：表格

  • MIND：脑图

  • FOLDER：文件夹

operatorId

String

是

用户的unionId，可调用查询用户详情接口获取。

parentNodeId

String

否

父节点Id，调用获取节点列表接口获取DentryModel参数值。如果不传，默认在根目录创建。

父节点可以是文件夹，也可以是普通的文档。

templateId

String

否

文档模板id。

templateType

String

否

文档模板类型，支持以下值：

  • public_template：推荐模板

  • team_template：团队模板

  • user_template：个人模板
