插入内容

调用本接口，在文档的指定位置插入一段内容，内容类型支持 Markdown。

接口调用说明

插入位置可以是文档根节点的children，也可以是支持插入块元素的 BlockElement（如高亮块、Table）的 children，通过 path和 index参数共同确定插入位置。
