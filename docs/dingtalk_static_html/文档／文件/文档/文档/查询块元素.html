查询块元素

调用本接口查询文档下的一级块元素，即根节点下的第一级的块元素，支持按起止位置和块类型进行过滤。

接口调用说明

如需要了解文档内容相关的数据结构，请参考文档数据结构内容。

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/doc/suites/documents/{docKey}/blocks

HTTP Method

GET

支持的应用类型

appType-企业内部应用

权限要求

permission-Storage.File.Read-企业存储文件读权限

名称

类型

String

是

调用该接口的访问凭证，通过调用获取企业内部应用的accessToken接口获取。

路径参数

名称

类型

是否必填

描述

docKey

String

是

文档ID（文档的 docKey 或 dentryUuid），通过创建知识库文档接口创建文档后，从返回值中获取docKey参数。

从文档URL获取dentryUuid，以https://alidocs.dingtalk.com/i/nodes/Zxxxxa-id为例，dentryUuid就是Zxxxxa-id。

查询参数

名称

类型

是否必填

Integer

endIndex

Integer

否

终止位置，为大于等于 0 的整数，表示查询到第endIndex个块元素为止。

blockType

String

否

指定块元素类型，详情参考块元素数据结构。

operatorId

String

是

用户unionId，可通过查询用户详情接口获取unionid参数值。
