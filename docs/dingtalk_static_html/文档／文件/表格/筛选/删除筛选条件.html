删除筛选条件

调用本接口可清除全局筛选中指定列的筛选条件，使该列不再参与筛选计算。适用于需要取消某一列的筛选规则但保留其他列筛选状态的场景，例如在多列筛选中临时解除某列的过滤限制以查看更多数据。

接口调用说明

接口调用成功后，之前因该列条件被隐藏的行将重新显示，前提是这些行未被其他列的筛选条件隐藏。
