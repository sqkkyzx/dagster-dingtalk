删除筛选视图

调用本接口可删除指定工作表上的某个筛选视图及其所有筛选条件。适用于清理不再需要的筛选视图的场景，例如在项目结束后移除临时创建的数据过滤视图，或在达到每个工作表 20 个筛选视图上限时删除旧视图以腾出空间。

接口调用说明

删除操作不会影响全局筛选和其他筛选视图。
