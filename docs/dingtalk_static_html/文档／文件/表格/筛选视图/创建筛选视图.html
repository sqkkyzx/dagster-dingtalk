创建筛选视图

调用本接口可在钉钉表格的指定工作表上创建筛选视图，并指定视图名称和筛选范围。创建成功后，可进一步为各列配置筛选条件。

接口调用说明

  • 筛选视图是独立于全局筛选的个人化数据过滤方式，适用于多人协作场景下不同成员需要查看不同数据子集的情况。

  • 每个工作表最多可创建 20 个筛选视图。

range

String

是

筛选视图的范围，使用 A1 表示法。例如 A1:D10 表示从 A1 到 D10 的区域。筛选范围的第一行将作为标题行，不参与筛选计算。

criteria

Map<String, Object>

否

创建时同时设置各列的筛选条件。key 为列偏移量（相对于 range 首列，从 0 开始），value 为筛选条件对象。不传则创建空筛选视图。

Object

否

当前列的筛选条件

filterType

String

否

筛选类型。可选值：

  • values：按值筛选

  • color：按颜色筛选

  • condition：按条件筛选

visibleValues

Array of String

否

可见值

conditions

Array

否

按条件筛选时，指定筛选条件列表。filterType 为 condition 时有效。每个条件对象包含 operator（操作符）和 value（条件值）字段。

operator

String

否

条件操作符。取值：

  • equal：等于

  • not-equal：不等于

  • contains：包含

  • not-contains：不包含

  • starts-with：开头是

  • not-starts-with：开头不是

  • ends-with：结尾是

  • not-ends-with：结尾不是

  • greater：大于

  • greater-equal：大于等于

  • less：小于

  • less-equal：小于等于

value

String

否

条件值

conditionOperator

String

否

多条件之间的逻辑关系。可选值：and（且）、or（或）。默认为 and。filterType 为 condition 且有多个条件时有效。

backgroundColor

String

否

按背景色筛选时，指定颜色值（十六进制，如 #FF0000）。filterType 为 color 时有效，与 fontColor 二选一。

fontColor

String

否

按字体色筛选时，指定颜色值（十六进制，如 #FF0000）。filterType 为 color 时有效，与 backgroundColor 二选一。
