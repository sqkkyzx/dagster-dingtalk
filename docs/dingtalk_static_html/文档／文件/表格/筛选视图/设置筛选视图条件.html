设置筛选视图条件

调用本接口可为指定筛选视图中的某一列设置或更新筛选条件，适用于在个人筛选视图中精确控制单列数据可见范围的场景，例如在个人工作视图中仅显示自己负责的项目，或按特定颜色标记筛选出高优先级任务。

接口调用说明

筛选视图的条件仅影响当前视图，不会改变其他用户看到的数据。
