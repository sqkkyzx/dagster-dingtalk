获取筛选视图列表

调用本接口可获取指定工作表上所有筛选视图的列表，返回每个筛选视图的 ID、名称、范围及筛选条件。适用于在管理或切换筛选视图前枚举当前已有的视图，或在程序中批量读取筛选视图配置以进行后续处理的场景。

criteria

Map<String, Object>

各列的筛选条件。key 为列偏移量（相对于 range 首列，从 0 开始），value 为筛选条件对象。无筛选条件时为空对象 {}。

Object

筛选条件

filterType

String

筛选类型。可选值：values（按值筛选）、condition（按条件筛选）、color（按颜色筛选）。

visibleValues

Array of String

可见值

conditions

Array

按条件筛选时，指定筛选条件列表。filterType 为 condition 时有效。每个条件对象包含 operator（操作符）和 value（条件值）字段。

operator

String

条件操作符。取值：

  • equal：等于

  • not-equal：不等于

  • contains：包含

  • not-contains：不包含

  • starts-with：开头是

  • not-starts-with：开头不是

  • ends-with：结尾是

  • not-ends-with：结尾不是

  • greater：大于

  • greater-equal：大于等于

  • less：小于

  • less-equal：小于等于

value

String

条件值

conditionOperator

String

多条件之间的逻辑关系。可选值：and（且）、or（或）。默认为 and。filterType 为 condition 且有多个条件时有效。

backgroundColor

String

按背景色筛选时，指定颜色值（十六进制，如 #FF0000）。filterType 为 color 时有效，与 fontColor 二选一。

fontColor

String

按字体色筛选时，指定颜色值（十六进制，如 #FF0000）。filterType 为 color 时有效，与 backgroundColor 二选一。
