设置自动行高

调用本接口，可根据指定的字体大小自动调整钉钉表格中指定行的行高，使内容完整显示。

接口调用说明

  • 根据传入的 fontWidth（字体大小）自动调整指定行范围内每行的行高，确保内容完整显示。

  • 通过 row（起始行游标）和 rowCount（行数）指定需要自动调整的行范围。

  • 行游标从 0 开始计数，即第一行的游标为 0。
