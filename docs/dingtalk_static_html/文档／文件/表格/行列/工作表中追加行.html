工作表中追加行

调用本接口，可在钉钉表格工作表的已有数据末尾追加若干行数据。系统会自动定位到最后一行有数据的位置，在其下方插入新行。

接口调用说明

  • 自动定位工作表中最后一行有数据的位置，在其下方追加新行数据。

  • 支持一次追加多行数据，通过 values 二维数组传入。

  • 如果工作表为空，数据将从第一行开始写入。

  • 返回追加数据所在的单元格范围（A1 表示法）。

  • 追加的数据列数应与工作表已有数据的列数保持一致，以确保数据对齐。
