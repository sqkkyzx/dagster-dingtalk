合并单元格

调用本接口，可将钉钉表格中指定区域内的多个单元格合并为一个整体。适用于制作表头跨列标题、合并相同分类行等需要调整单元格布局的场景。支持全部合并、按行合并、按列合并三种模式，合并后仅保留左上角单元格的数据，其余单元格内容将被清除，接口返回实际操作的单元格地址。

mergeType

String

否

指定合并单元格的方式。可选值：

  • mergeAll：合并所有单元格（默认值）

  • mergeRows：按行合并，即在选定的区域内，将同一行相邻的单元格合并成一个单元格

  • mergeColumns：按列合并，即在选定的区域内，将同一列中相邻的单元格合并成一个单元格
