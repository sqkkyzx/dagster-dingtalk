创建浮动图片

调用本接口，可获取钉钉表格指定工作表中所有浮动图片的列表信息。适用于需要批量读取工作表内浮动图片的场景，如导出、同步或审计等。返回结果包含每张图片的唯一标识、图片地址、锚点位置及尺寸偏移信息。
