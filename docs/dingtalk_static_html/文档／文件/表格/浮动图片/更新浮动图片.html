更新浮动图片

调用本接口，可更新钉钉表格中已有浮动图片的属性，包括图片地址、锚点位置和尺寸偏移量。支持部分更新，未传入的字段将保留原值，例如只传 `anchor` 即可单独调整图片位置而不影响其他属性。适用于调整图片位置、更换图片源或修改展示尺寸等场景。成功后返回更新后的图片完整属性。
