概述

本文档介绍什么是钉钉运动，如何开通钉钉运动，钉钉运动接口能力。

什么是钉钉运动

钉钉运动是一款钉钉官方应用，用于记录分享企业员工行走步数，以及生成排行榜，丰富企业员工交流互动。

如何开通钉钉运动

钉钉运动只支持钉钉手机端使用。

手机端：钉钉手机客户端-工作台

如何参与企业步数排行榜

只能通过钉钉手机端操作：

  1. 在钉钉工作台页面，单击钉钉运动。

  2. 在步数排行榜页面，单击右上角更多，单击设置。

  3. 在设置页面，找到对应企业，开启加入步数排行榜。

开放概览

钉钉运动提供了丰富的接口开放能力，开发者通过API接口可以实现钉钉运动和企业业务系统打通。
