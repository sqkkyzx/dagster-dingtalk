查询用户是否参与企业步数排行榜

调用本接口，查询用户是否参与企业的步数排行榜。

请求

基本信息

HTTP URL

https://oapi.dingtalk.com/topapi/health/stepinfo/getuserstatus

HTTP Method

POST

支持的应用类型

appType-企业内部应用appType-第三方企业应用

权限要求

permission-qyapi_health_read-钉钉用户运动步数查询权限

名称

类型

是否必填

  • 第三方企业应用，通过获取第三方企业的access_token接口获取。
