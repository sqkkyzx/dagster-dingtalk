获取群会话的OpenConversationId

调用本接口，通过chatId查询OpenConversationId，适用于在创建群会话时未保存 OpenConversationId，后续业务需要使用该 ID 的场景。

接口调用说明

如果在调用创建群接口时没有保存OpenConversationId，可以通过调用本接口通过chatId获取OpenConversationId。
