查询群简要信息

调用本接口，根据群ID查询群名称、群图标、群主id等基本信息，适用于需要快速获取群基本信息的场景，如在群列表展示群名称、头像等信息，或者在业务处理中需要判断群状态等情况。

接口调用说明

支持以下场景使用：

  • 基于群模板创建的群，详情参见创建场景群。

  • 安装群聊酷应用的群，详情参见群聊酷应用。
