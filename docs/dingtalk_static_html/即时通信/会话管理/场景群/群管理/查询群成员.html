  • 安装群聊酷应用的群，通过安装酷应用入群获取返回参数openConversationId参数值。

Long

接口返回结果可能会大于或小于maxResults，以实际返回结果为准。如果群成员数量不超过1000，而直接一次性返回全部群成员；如果群成员数量大于1000，则按照分页大小分批次返回。

nextToken

String
