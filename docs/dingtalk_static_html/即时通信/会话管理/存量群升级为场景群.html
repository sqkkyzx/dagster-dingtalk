存量群升级为场景群

如果你需要将存量群升级为场景群，可以参考本文档操作步骤。

前提条件

完成创建群模板的流程。

操作步骤

  1. 登录开发者后台，单击顶部导航栏开放能力 > 场景群。选择侧边导航栏群模板。

  2. 选择目标群模板，单击目标群模板右下角灰度，填写企业CorpId。

  3. 存量群升级场景群：

    • 调用启用群模板接口，你可以将存量群升级为场景群。

    • 你可以登录钉钉客户端：

      1. 找到目标群会话，单击右上角群设置标识。

      2. 单击群管理 > 快捷栏管理 > 管理酷应用，进入酷应用页面。

      3. 单击最下方群模板设置，选择上述目标群模板开启即可。

如果你需要通过接口直接创建场景群，请参考创建群接口。
