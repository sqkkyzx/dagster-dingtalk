创建并投放卡片

调用本接口，可以创建卡片实例，并将卡片投放至多个指定场域，创建卡片的时候，除了设置卡片的基本数据，还可以设置动态数据源等。

接口调用说明

调用说明

说明

目前支持将卡片投放至以下场域：IM群聊、IM单聊酷应用、IM机器人单聊、吊顶。

在将卡片投放到不同的场域时，使用outTrackId唯一标识一张卡片，通过openSpaceId标识需要被投放的场域及其场域Id，通过openDeliverModels传入不同的投放场域。
