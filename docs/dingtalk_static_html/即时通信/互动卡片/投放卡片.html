投放卡片

调用本接口实现多个指定场域的卡片投放。

接口调用说明

在将卡片投放到不同的场域时，使用outTrackId唯一标识一张卡片，通过openSpaceId标识需要被投放的场域及其场域Id，通过openDeliverModels传入不同场域下的投放属性。

说明

  • 目前支持将卡片投放至以下场域：IM群聊、IM单聊酷应用、IM机器人单聊、吊顶、协作、文档。

  • 若需要被投放的卡片实例不支持该场域，需要先调用appendSpace接口增加该场域（将卡片投放至文档不需要）。
