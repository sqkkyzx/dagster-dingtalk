下载机器人接收消息的文件内容

获取机器人消息中文件下载链接

接口调用说明

  1. 当用户向机器人发送图片消息、语音消息、视频消息、文件消息、富文本消息时，通过机器人回调给开发者的信息中获取临时下载码downloadCode。

  2. 通过临时下载码downloadCode，调用本接口获取消息中的图片、语音、文件、视频等文件的临时下载链接。

  3. 访问临时下载链接，获取文件内容。

说明

  • 群聊会话中：群成员@机器人，机器人不支持接收语音、文件、视频类型。

  • 人与人的会话中：机器人不支持接收语音、文件、视频类型。

  • 人与机器人的会话中：机器人支持接收语音、文件、视频类型。
