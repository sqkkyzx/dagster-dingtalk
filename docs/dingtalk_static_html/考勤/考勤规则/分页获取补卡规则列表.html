分页获取补卡规则列表

调用本接口，分页获取考勤打卡中设置的补卡规则列表，包括规则名称和规则ID。

接口调用说明

例如，企业的考勤规则管理内有两个补卡规则，分别为测试补卡规则和默认补卡规则，如下图所示。

调用本接口可获取两个补卡规则的规则名称和规则ID。
