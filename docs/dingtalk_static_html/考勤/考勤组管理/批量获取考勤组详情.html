批量获取考勤组详情

调用本接口，查询所有的考勤组详情信息。

接口调用说明

在钉钉考勤应用中，考勤组是一类具有相同的班次、考勤位置等考勤规则的人或部门的组合，企业可根据实际业务设置多个考勤组。

说明

本接口与获取考勤组详情接口的区别，该接口可以获取包含考勤组内的班次设置相关信息。

请求

基本信息

HTTP URL

https://oapi.dingtalk.com/topapi/attendance/getsimplegroups

HTTP Method

POST

权限要求
