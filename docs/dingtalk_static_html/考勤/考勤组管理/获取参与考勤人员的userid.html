获取参与考勤人员的userid

调用本接口，分页获取某个考勤组下的所有员工的userId。

接口调用说明

如果参与考勤人员设置了部门，接口返回部门下的员工的userId，不返回部门ID。

例如，参与考勤人员设置了测试部门、员工张三，测试部门下有一个员工李四。调用本接口可以获取到如下信息：

  • 测试部门（包括子部门）下的员工的userId，即李四的userId

  • 员工张三的员工userId
