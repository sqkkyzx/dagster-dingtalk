更新考勤组

调用本接口，根据考勤组ID更新考勤组信息。

请求

基本信息

HTTP URL

https://oapi.dingtalk.com/topapi/attendance/group/modify

HTTP Method

POST

支持的应用类型

appType-企业内部应用

权限要求

permission-qyapi_attendance_group_manage-考勤组管理权限
