创建班次

调用本接口，创建钉钉考勤班次。

请求

基本信息

HTTP URL

https://oapi.dingtalk.com/topapi/attendance/shift/add

HTTP Method

POST

支持的应用类型

appType-企业内部应用

权限要求

permission-qyapi_attendance_group_manage-考勤组管理权限
