获取报表假期数据

调用本接口，根据假期名称和用户ID获取钉钉智能考勤报表的假期数据，其中包含了一定时间段内报表假期列的所有数据，由于假期列是一个动态列，因此需要根据假期名称获取数据。

接口调用说明

更多数据开放及消费能力请移至数据资产平台。数据资产平台（dPaaS）是为企业提供的统一数据管理平台，基于钉钉构建安全、可扩展、易维护和管理的数据服务，助力业务决策！
