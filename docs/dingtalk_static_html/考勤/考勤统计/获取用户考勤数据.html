获取用户考勤数据

调用本接口，可获取用户的考勤数据，包括打卡流水记录、打卡结果和审批列表等。

请求

基本信息

HTTP URL

https://oapi.dingtalk.com/topapi/attendance/getupdatedata

HTTP Method

POST

支持的应用类型

appType-企业内部应用

权限要求

permission-qyapi_get_attendance_data-考勤数据读权限

是否必填
