获取微应用后台免登的accessToken

调用本接口获取微应用后台免登的accessToken。获取的accessToken只在应用管理后台免登场景中使用。

请求

基本信息

HTTP URL

https://api.dingtalk.com/v1.0/oauth2/ssoAccessToken

HTTP Method

POST

支持的应用类型

appType-企业内部应用appType-第三方企业应用

权限要求

permission-qyapi_base-调用企业API时需要具备的基本权限

请求体

名称

类型

是否必填

示例值

描述

corpid

String

是

corpxxxx

企业的corpId。

ssoSecret

String

是

shYgsk7xxx

sso密钥，可以在开发者后台基本信息—开发信息（旧版）页面查看。

请求示例

curl -X POST 'https://api.dingtalk.com/v1.0/oauth2/ssoAccessToken' \
--H 'Content-Type: application/json' \
--d '{
    "corpid": "corpxxxx",
