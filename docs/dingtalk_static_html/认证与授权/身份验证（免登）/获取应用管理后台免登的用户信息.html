获取应用管理后台免登的用户信息

在应用管理后台免登场景中，需要本接口通过获取到的免登授权码code和获取到的应用后台免登的access_token来换取应用管理员的身份信息。

请求

基本信息

HTTP URL

https://api.dingtalk.com/v1.0/oauth2/ssoUserInfo

HTTP Method

GET

支持的应用类型

appType-企业内部应用appType-第三方企业应用

权限要求

permission-qyapi_get_omp_sso_userinfo-企业OA后台免登访问权限

名称

类型

名称

类型

是否必填
