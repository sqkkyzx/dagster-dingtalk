设置部门可见性优先级

调用本接口，设置通讯录部门可见性优先级。

接口调用说明

如下图所示，在默认情况下子部门（高管）部门成员对通讯录查看范围虽然与其父部门不同，但是仍会沿用父部门的配置。

如果需要使子部门（高管）单独设置的部门成员对通讯录查看范围生效，可调用本接口将子部门的优先级进行提升。

部门设置的优先级默认为：父部门设置优先于子部门。
