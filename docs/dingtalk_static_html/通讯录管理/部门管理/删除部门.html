删除部门

调用本接口根据部门ID删除指定部门。

接口调用说明

以下情况无法删除部门：

  • 不能删除根部门，即部门ID为1。

  • 部门或子部门内还有未删除的员工。

请求

基本信息

HTTP URL

https://oapi.dingtalk.com/topapi/v2/department/delete

HTTP Method

POST

支持的应用类型

appType-企业内部应用

权限要求

permission-qyapi_manage_addresslist-通讯录数据管理权限
