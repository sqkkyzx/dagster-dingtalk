获取异步转译任务结果

调用本接口，获取查询已经提交过的转译任务结果。

接口调用说明

接口功能介绍

如果第三方企业应用选择了通讯录加密部署方式，该应用调用通讯录等相关接口，无法获取授权企业员工姓名、职位和所在部门名称等信息。可以通过以下方法获取：

例如，某企业授权开通一个第三方企业应用，使用应用导出本企业可以使用该第三方应用的员工列表及信息，包含员工userID、员工姓名、员工所在部门ID和员工所在部门名称。可按照以下步骤实现。

  1. 构造需要转译的文档，示例如下图所示。

    说明

    只支持xlsx，xls，csv，txt格式文件。

  2. 调用单步文件上传接口，获取media_id。

  3. 调用异步转译通讯录ID接口，获取转译任务ID。

    说明

    调用异步转译通讯录ID接口后，如果文件转译成功，会触发异步任务回调通知。

  4. 调用本接口，获取转译后的文件下载链接。

    说明

    返回的文件下载链接建议使用<a>标签引用，download属性不可指定（浏览器兼容性问题）。

  5. 用户下载得到的文件内容，如下图所示。
