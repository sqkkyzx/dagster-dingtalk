异步转译通讯录ID

调用本接口，发起异步通讯录ID内容转译，替换产品方案商通讯录权限范围内的用户ID和部门ID。

接口调用说明

如果第三方企业应用选择了通讯录加密部署方式，该应用调用通讯录等相关接口，无法获取授权企业员工姓名、职位和所在部门名称等信息。可以通过以下方法获取：

例如，某企业授权开通一个第三方企业应用，使用应用导出本企业可以使用该第三方应用的员工列表及信息，包含员工userID、员工姓名、员工所在部门ID和员工所在部门名称。可按照以下步骤实现。

  2. 调用单步文件上传接口，获取media_id。

  3. 调用本接口，获取转译任务ID。

    说明

在调用本接口的过程中，你可能会遇到以下问题：

  • 问题：使用NodeJs第三方库exceljs生成的excel文件，文件内容看上去正常，但是转译不成功。

  • 问题原因：exceljs在不开启sharedString的情况下，会默认使用formula string进行文件内容写入，这种行为不合规范，导致转译失败。

  • 解决方法：查看exceljs文档，强制开启useSharedStrings选项。
