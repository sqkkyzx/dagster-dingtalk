设置高管模式

调用本接口设置员工的高管模式。

接口调用说明

当设置员工为高管，则针对设置的拦截场景，同企业内只有其他高管、高管白名单内用户以及高管的好友可以联系。其他人不可联系。 例如：设置userA为高管，拦截场景为VIDEO_CALL，白名单用户设置了userB。userB、userC为同一企业的普通员工，且跟userA没有好友关系，则userB可以对userA发起视频通话，userC不可以对userA发起视频通话。
