删除用户

调用本接口根据用户的userid删除指定用户。

接口调用说明

调用本接口成功删除通讯录员工，同时会删除以下内容：

  • 该员工所在的考勤组会同步删除该员工。

  • 智能人事花名册会同步删除该员工，员工信息可以在智能人事已离职人员中查看。

基本信息

https://oapi.dingtalk.com/topapi/v2/user/delete

HTTP Method
