查询企业账号状态

调用本接口，查询某企业账号的启用状态。

请求

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/contact/orgAccounts/status

HTTP Method

GET

支持的应用类型

appType-企业内部应用

权限要求

permission-Contact.OrgAccountSecurity.ReadWrite-通讯录企业账号安全数据读写权限

名称

类型

String

是

  • 创建钉钉自建企业账号

  • 邀请其他组织企业账号加入
