企业账号转交主管理员（创建者）

调用本接口，将本组织内某企业账号有所有权的组织，转交给另一企业账号，如果接收的账号不在该组织内则自动加入。

接口调用说明

例如，钉钉组织A下有两个企业账号，分别为企业账号A1和企业账号A2，其中企业账号A1是另一企业名称为钉钉组织B的创建者身份（当前测试企业OA管理后台-安全与权限-主管理员列表可查看）。调用本接口可用于把钉钉组织B的创建者身份变更为企业账号A2。
