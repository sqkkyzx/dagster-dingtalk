授权企业账号可加入多组织

调用本接口，授权企业账号可以加入多个组织，只有被授权后企业账号才能加入外部组织。

接口调用说明

授权成功后，只支持接口操作添加进入企业。

请求

基本信息

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/contact/orgAccounts/multiOrgPermissions/auth

HTTP Method

POST

支持的应用类型

appType-企业内部应用　

权限要求

permission-Contact.OrgAccountSecurity.ReadWrite-通讯录企业账号安全数据读写权限

请求头

名称

类型

是否必填

描述

String

是

调用该接口的访问凭证，通过调用获取企业内部应用的accessToken接口获取。

请求体

名称

类型

是否必填

描述

String

grantDeptIdList

Array of Long

否

授权的部门列表。

  • 如果该参数不传，授权整个企业。

  • 如果该参数传值，授权参数值对应的部门。
