企业账号修改钉钉号

调用本接口，修改企业账号钉钉号。

接口调用说明

每个账号修改钉钉号后，一年内不能再次修改。

请求

基本信息

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/contact/orgAccounts/dingTalkIds/change

HTTP Method

POST

支持的应用类型

appType-企业内部应用

权限要求

permission-qyapi_manage_addresslist-维护通讯录的接口访问权限

请求头

名称

类型

是否必填

描述

String

是

调用该接口的访问凭证，通过调用获取企业内部应用的accessToken接口获取。

请求体

名称

类型

是否必填

描述

userId

String

是

员工userId，只支持归属于本企业的企业账号userId。

dingTalkId

String

是

新的钉钉号。全局唯一，不只是组织内唯一，所以不能和已经存在的冲突，发生冲突时需要更换备选值。格式要求：

  • 6<=长度<=20

  • 字母开头

  • 只包含字母、数字

  • 不能包含违禁内容
