授权其他组织查看本组织的企业账号信息

授权当前组织的企业账号在加入其他组织之后，在其他组织查看组织的企业账号信息的具体字段

请求

基本信息

字段

值

HTTP URL

https://api.dingtalk.com/v1.0/contact/orgAccounts/mobiles/visibleInOtherOrg

HTTP Method

PUT

支持的应用类型

appType-企业内部应用

权限要求

permission-Contact.OrgAccount.VisiblityInOtherOrg-企业账号加入其他组织后数据可见性权限

请求头

名称

类型

是否必填

描述

x-acs-dingtalk-access-token

String

是

调用该接口的访问凭证，通过调用获取企业内部应用的accessToken接口获取。

请求体

名称

类型

是否必填

描述

toCorpIds

Array of String

是

被授权组织对应corpId列表。

被授权组织的corpId列表和调用接口的调用者对应 userId，都不能为空且参数出错会导致调用报错。

optUserId

String

是

当前调用接口的调用者对应userId。

用于数据审计，需填写当前组织下真实的员工userId，否则接口调用会失败。

fields

Array of String

否

授权字段:

如果fields为空或者空集合表示仅授权mobile（历史版本兼容）

  • mobile：企业账号手机号。

  • status： 企业账号停用状态。
