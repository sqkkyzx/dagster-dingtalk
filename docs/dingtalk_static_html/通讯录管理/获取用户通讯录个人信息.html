获取用户通讯录个人信息

调用本接口获取企业用户通讯录中的个人信息。

接口调用说明

企业内部应用和第三方企业应用在调用该接口前，需要先获取个人用户的accessToken，获取方式请参考获取登录用户的访问凭证。

请求

基本信息

HTTP URL

https://api.dingtalk.com/v1.0/contact/users/{unionId}

HTTP Method

GET

支持的应用类型

appType-企业内部应用appType-第三方企业应用appType-第三方个人应用

权限要求

permission-Contact.User.Read-通讯录个人信息读权限

名称

类型

是否必填

描述

String

Path参数

名称

类型

是否必填
