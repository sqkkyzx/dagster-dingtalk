更新外部联系人

调用本接口，更新企业外部联系人。

请求

基本信息

https://oapi.dingtalk.com/topapi/extcontact/update

HTTP Method

POST

appType-企业内部应用

权限要求

permission-qyapi_ext_write-企业外部联系人写权限
