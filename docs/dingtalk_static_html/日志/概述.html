概述

本文介绍了日志产品使用，如何开通日志应用等内容。

什么是日志

方便管理者了解员工每日工作情况，可以帮助员工总结沉淀工作经验。更多介绍请参见钉钉使用手册-日志。

如何开通日志

开发者可以通过钉钉移动端或钉钉PC端使用日志。

手机端：钉钉移动端-工作台

电脑端：钉钉PC端-工作台

开放概览

日志提供了丰富的接口开放能力，开发者通过API接口可以实现日志和企业自由的业务系统打通。

名词解释

日志模板标识字段（report_code）

日志模板标识字段，定义为report_code。例如周报、日报都属于日志模板，都有分别对应的report_code值。

日志内容（contents）

日志内容，包含日志内每一个组件的排序、类型、文本内容等属性信息，定义为contents。

日志唯一标识字段（report_id）

每篇日志对应唯一标识字段report_id。
