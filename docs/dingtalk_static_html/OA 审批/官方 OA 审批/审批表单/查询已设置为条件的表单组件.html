用于查询已设置为条件的表单组件。

接口调用说明

产品方案商通过创建审批模板接口，帮企业生成审批模板后，企业可以在审批管理后台设置审批流程。设置过程中，可以把表单组件设为流程条件。此时，这些被设为流程条件的表单组件，ISV是不能再次更新的。通过此接口，可以查询有哪些表单组件被设置为流程条件了。
