授权预览审批附件

调用本接口，授权预览审批附件。

接口调用说明

说明

本接口需配合钉盘JSAPI使用。

  1. 获取审批钉盘空间spaceId，调用获取审批钉盘空间信息接口，获取审批钉盘空间spaceId。

  2. 根据审批钉盘空间spaceId，网页应用（H5微应用）/小程序，通过上传附件到钉盘/从钉盘选择文件获取钉盘附件file的信息。

  3. 获取审批实例processInstanceId，调用发起审批实例接口获取InstanceId参数值。

  4. 根据上述获取信息，调用本接口，授权用户审批附件预览权限。每一次预览审批附件前，都需要调用本接口进行授权。

  5. 调用预览钉盘文件接口，进行预览，调用预览钉盘文件接口。
