获取企业内部小程序的版本列表

调用本接口，获取指定企业内部小程序的所有版本信息，包括版本ID、版本号、版本类型、创建时间、更新时间等。

接口调用说明

仅企业内部自建小程序支持调用该接口获取版本数据。
